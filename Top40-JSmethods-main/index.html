Array

You may know that a variable can store only a value at a time, for example var personName = "joey", this is totally fine and we may use it many times while building a project, however, sometimes it is required to collect many values in a single variable like a list of students names, this is where we can use the Array concept. Array is a single variable that store a list of values and each element is specified by a single index.
